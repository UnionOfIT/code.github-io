Как работает шифрование на этом сайте?

Этот сайт позволяет зашифровать и расшифровать текст с помощью одного или двух паролей. Шифрование происходит в несколько этапов, чтобы максимально усложнить взлом.

Этапы шифрования

  1. Виженер (по первому паролю): Для текста генерируется уникальный ключ (3-10 символов, без подряд одинаковых, буквы/цифры/символы), который строится детерминированно на основе пароля и длины текста. Текст шифруется этим ключом.
  2. Цезарь (по первому паролю): Для каждой буквы выбирается свой сдвиг (от 1 до 22), который зависит от пароля. Это делает перебор невозможным.
  3. Атбаш: Каждая буква заменяется на «зеркальную» относительно алфавита (например, «а» → «я»).
  4. Виженер (по второму паролю, если введён): Если введён второй пароль, применяется ещё один шифр Виженера с уникальным ключом по второму паролю.
  5. Двоичный код: Итоговый текст переводится в двоичный код (каждая буква — 8 бит).

Этапы расшифровки

  1. Двоичный код переводится обратно в текст.
  2. Если был второй пароль — обратный Виженер по второму паролю.
  3. Обратный Атбаш.
  4. Обратный Цезарь (по первому паролю).
  5. Обратный Виженер (по первому паролю).

Безопасность

  • Ключи для Виженера генерируются так, что даже зная схему, но не зная паролей, подобрать их невозможно.
  • Два пароля — можно использовать один или оба для максимальной защиты.
  • Многоступенчатость делает анализ и взлом крайне сложным.
  • Двоичный код скрывает структуру текста.

Для чего подходит

  • Для личных секретов, переписки, игр, передачи скрытых сообщений.
  • Для защиты от обычных людей, друзей, учителей, родителей и т.д.
  • Не подходит для хранения государственных или банковских тайн.
